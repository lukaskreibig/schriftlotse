SchriftLotse

Alte Schriften. Klar gelesen.

Lokal und offen: Historische deutschsprachige Scans entziffern, prüfen, exportieren und über alle Dokumente durchsuchen.

Funktionen

Adaptive Bildaufbereitung, freie OCR-/HTR-Modelle, lokaler CHURRO-Zweitleser, Stapelverarbeitung, originalgetreue Transkription, Lesefassung, PDF/DOCX/TXT/PAGE-XML/ALTO, eScriptorium-Austausch und eine fehlertolerante Archivsuche mit Namensvarianten und Bedeutungsähnlichkeit.

Installation

Das öffentliche Repository enthält nur Code und Dokumentation. Modelle und Dokumente bleiben lokal. Lade das Projekt von GitHub und öffne SchriftLotse.command.

Nach der Ersteinrichtung erzeugt scripts/build_macos_app.sh außerdem eine native SchriftLotse.app, die den lokalen Dienst in einem eigenen App-Fenster öffnet.

Datenschutz

Die Oberfläche bindet ausschließlich an 127.0.0.1. OpenRouter erhält nur einen ausdrücklich ausgewählten Ausschnitt. Private Scans gehören niemals auf GitHub.
